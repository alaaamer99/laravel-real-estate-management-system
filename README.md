# 🏢 Laravel Real Estate Management System
Ready-to-use Laravel real estate management system to manage properties, clients, and sales/rent deals — perfect for agencies and freelancers.

---

## 🔥 Features

* Property Management (Sale / Rent)
* Client Management
* Deals Tracking
* Admin & Agent Roles
* Clean Dashboard

---

## 🎯 Perfect For

* Real estate companies
* Property agencies
* Freelancers building CRM systems

---

## ⚡ Demo Scenario

* Add properties (apartments, villas, shops)
* Assign clients
* Track sales and rental deals

---

## 🚀 Quick Start

```bash
git clone ...
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
```
